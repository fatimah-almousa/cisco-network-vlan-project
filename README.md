Cisco Network VLAN Project

This project was created for the IT Infrastructure Management (CIS 326) course. It focuses on designing and simulating a small office network using Cisco Packet Tracer.

Project Idea

The goal of this project is to build a working network for an office environment (LawEase Office Network). The network connects multiple floors using routers and switches, and separates traffic using VLANs.

What was done

* Set up routers and switches for each floor
* Created VLANs and assigned switch ports
* Configured basic routing between networks
* Built a simple office topology using Cisco Packet Tracer
* Tested connectivity between devices

Tools Used

* Cisco Packet Tracer
* Basic networking commands (CLI)
* VLAN configuration

Files

* fileee.pkt – Packet Tracer project file
* report.pdf – Project report (if included)

What I learned

* How VLANs work and why they are useful
* How switches and routers communicate
* Basics of designing a structured network
* Using Cisco Packet Tracer for simulation
